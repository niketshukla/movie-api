MyFlix

The web application will provide users with access to information about different movies, directors, and genres.

Request URL - Endpoint Query Parameter Http Method Request Data Format Response
Return a list of all movies to the user /movies None GET None A JSON object holding data about all the movies
Return data about a single movie by title to the user /movies/:title title GET None A JSON object holding data about a single movie based on the title passed in the url containing (description, genre, director, image Url)
Return data about a genre by name/title (eg. Thriller) /movies/genre/:genreName genreName GET None A JSON object of movies based on genre passed in the url
Return data about a director by name /movies/director/:directorName directorName GET None A JSON object of movies based on director name passed in the url
Allow new users to register themselves /users None POST A JSON object about a user to be registered A Json object about the user that is added with user id
Allow users to update their info ie. username /users/:name name PUT None Returns a success message that the username has been updated
Allow users to add movie to their favorites /users/:favorites favorites PUT None Returns a success message that the movie added to favorites
Allow users to remove movie from their favorites /users/:favorites favorites DELETE None Returns a success message that the movie removed from favorites
Allow users to deregister /users None DELETE None Returns a success message that the user is removed
